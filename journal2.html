第二篇日志

    新学习了javascript的内容，记录以下小点：
  • var：所谓变量提升，相当于使用前已经定义了此量位undifined
    let：块级作用域和c++的变量定义差不多，在使用前必须定义好。 在函数中、局部变量中也能和全局分开使用。有暂存死区的特性， 这一性质体现的就是先定义好这一点，和c++不同的是在块中也可 以定义和全局变量同名的变量，但是不可在定义前使用 （如用var，则适应var的特性）。
  • 关于生成器generator和yield:一般定义generator如: 
    
            function *foo(a)
            {
            for(let i = 0;i < len;++i)
              {
               if(a[i] === some 类型)yield a[i];
               else{
                  yield* a[i];//前提是啊a[i]和a的类型一致，如多叉树
                }
              }
            }
  • 关于boolean类型：&&和||同普通逻辑方式相似（如&&前为true取决于后者）
  • number类型：注意有不同的精度，影响===的关系。
